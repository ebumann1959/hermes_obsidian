## Accents — Priority Issue
This is your biggest weak point. Accents change meaning in Spanish. Missing them = wrong word or wrong grammar.
- "esta" ≠ "está" (is) — always check when you write "esta"
- "vamanos" ≠ "vámonos" (let's go) — if in doubt, add the accent
- "que paso" ≠ "¿Qué pasó?" (what happened) — question mark + accent on both
- "si" ≠ "sí" (if vs yes) — will come up later
- **Rule:** if a vowel feels stressed in a word, it probably needs an accent. In doubt, look it up.
