# Welcome to the Obsidian Vault

Created: 2026-04-15

This vault is shared across all Hermes agents on this system.

## Purpose
- Shared knowledge base for multi-agent coordination
- Project notes, decisions, and context
- Human-readable alongside agent memory

## Structure
- Use folders to organize by domain (e.g. `paperclip/`, `unity/`, `persona/`)
- Link notes with [[wikilinks]]

## Access
Agents read/write via filesystem at `/mnt/ssd/obsidian-vault/`
Vault path env: `OBSIDIAN_VAULT_PATH`
